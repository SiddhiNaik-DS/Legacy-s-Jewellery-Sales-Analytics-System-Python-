## ðŸ’» Local Setup and Project Launch Guide

This guide assumes you are starting from scratch on your local machine.

### 1\. Install Necessary Tools

  * **Python:** Ensure you have **Python 3.8 or newer** installed.
  * **Git:** Download and install the **Git command-line tool**.
  * **Visual Studio Code (VS Code):** We recommend VS Code for editing Python projects. Download and install it.

### 2\. Clone the Repository

Open your **Terminal** (macOS/Linux) or **Command Prompt/PowerShell** (Windows) and run the following command to download the project:

```bash
git clone [Your Repository URL Here]
cd "Legacy's Sales System"
```

### 3\. Create a Virtual Environment

It's best practice to keep project dependencies isolated. Run these commands from within the project folder:

```bash
# Create a virtual environment named 'venv'
python -m venv venv

# Activate the virtual environment
# On Windows (PowerShell):
.\venv\Scripts\Activate.ps1
# On macOS/Linux:
source venv/bin/activate
```

### 4\. Install Dependencies

Install all the required Python libraries (Streamlit, Pandas, Plotly, etc.):

```bash
pip install -r requirements.txt
```

*(**Note**: If you don't have a `requirements.txt` file yet, run `pip freeze > requirements.txt` now, and then tell users to install the dependencies.)*

### 5\. Run the Application

With the virtual environment active, run the Streamlit app:

```bash
streamlit run "Legacy's Sales System.py"
```

The application will automatically open in your web browser, usually at `http://localhost:8501`.

### 6\. Start Analysis

  * Navigate to the **"Historical Data Uploader & Summary"** page.
  * Click **"Browse Files"** to upload your `Sales Dataset.csv`.
  * Proceed through the remaining pages (**Details View** and **Time-Series Analytics**) to interact with the dashboard\!
